# How To Use Translated Strings

Translated strings can be retrieved using the [ApplicationIntlContext](). An `ApplicationIntlContext.Provider` is configured and provided by [ApplicationBase]().

```jsx
import React, { useContext } from 'react';
import { ApplicationIntlContext } from 'terra-application/lib/application-intl';

const ExampleComponent = () => {
  const applicationIntl = useContext(ApplicationIntlContext);

  return (
    <div>
      <p>This component renders a translated string</p>
      <p>{applicationIntl.formatMessage({ id: 'test.string.id' })}</p>
    </div>
  );
});

export default ExampleComponent;
```

> Note: The version of react-intl required by terra-application (v2.x) is **not** the latest version. The 2.x version of react-intl uses React's legacy context API. While it is possible to use the legacy context directly, it is **strongly** recommended to use the [ApplicationIntlContext]() for accessing intl data. When the react-intl version is updated in the future and the legacy support is dropped, consumers of the [ApplicationIntlContext]() will continue to function without further implementation changes.
