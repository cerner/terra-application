# How To Render Loading Overlays

Loading overlays can be rendered within the framework by rendering an [ApplicationLoadingOverlay](/components/terra-application/application/reference/components/application-loading-overlay) component. Overlays will be rendered by the closest ancestor [ApplicationLoadingOverlayProvider](/components/terra-application/application/reference/components/application-loading-overlay). All content rendered within the responsding ApplicationLoadingOverlayProvider will be overlaid.

```jsx
import React, { useState } from 'react';
import ApplicationLoadingOverlay from 'terra-application/lib/application-loading-overlay';

const ExampleComponent = () => {
  const [isLoading, setIsLoading] = useState(false);

  return (
    <div>
      <p>This component toggles a loading state that lasts for 3 seconds.</p>
      <p>
        <button type="button" onClick={() => {
          setIsLoading(true);

          setTimeout(() => {
            setIsLoading(false);
          }, 3000);
        }}>
          Start Loading
        </button>
      </p>
      <ApplicationLoadingOverlay isOpen={isLoading} />
    </div>
  );
};

export default ExampleComponent;
```

Any component that wants to create an overlay container for its children can render its own [ApplicationLoadingOverlayProvider](/components/terra-application/application/reference/components/application-loading-overlay) to scope overlay presentation to that container.

> Note: ApplicationLoadingOverlayProviders are provided by [ApplicationBase](/components/terra-application/application/reference/components/application-base), [ModalManager](/components/terra-application/application/reference/components/modal-manager), and the [SlidePanelManager](/components/terra-application/application/reference/components/slide-panel-manager).
