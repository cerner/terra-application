# How To Use Translated Strings

Translated strings can be retrieved using the [ApplicationIntlContext](/components/terra-application/application/reference/contexts/application-intl-context). An `ApplicationIntlContext.Provider` is configured and provided by [ApplicationBase](/components/terra-application/application/reference/components/application-base). The strings will be translated based upon the `locale` prop provided to ApplicationBase.

In addition to the ApplicationIntlContext, the components provided by react-intl can be used directly. However, the ApplicationIntlContext can be used by Hooks-based components to access i18n capabilities without having to use the `injectIntl` HOC-generator from react-intl.

```jsx
import React, { useContext } from 'react';
import { ApplicationIntlContext } from 'terra-application/lib/application-intl';
import { FormattedMessage } from 'react-intl';

const ExampleComponent = () => {
  const applicationIntl = useContext(ApplicationIntlContext);

  return (
    <div>
      <p>This component renders a translated string</p>
      <p>{applicationIntl.formatMessage({ id: 'test.string.id' })}</p>
      <p>The FormattedMessage component will render the same string</p>
      <FormattedMessage id='test.string.id' />
    </div>
  );
});

export default ExampleComponent;
```
